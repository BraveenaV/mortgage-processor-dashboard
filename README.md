# Mortgage Processor Dashboard
A responsive web dashboard designed for mortgage processors to manage document indexing and 'Clear-to-Close' workflows

## Overview
A front-end application designed to solve common inefficiencies in the mortgage closing process. [cite_start]This dashboard helps processors track document readiness for **FHA, VA, and Conventional loans**[cite: 18].

## Features
- **Dynamic Status Tracking:** Real-time visual indicators for "Clear-to-Close" vs "Pending Review" status.
- [cite_start]**Document Verification List:** A specialized UI for indexing Appraisal and Title documents[cite: 29, 30].
- **Responsive Design:** Fully optimized for desktop and tablet views using modern CSS.

## Tech Stack
- **HTML5/CSS3:** Semantic structure and responsive styling.
- **JavaScript:** Dynamic DOM manipulation for status updates.
- **UI/UX Principles:** Designed based on user-journey mapping for mortgage operations.
